# מיליונים 360 — סימולציה פיננסית אינטראקטיבית

כלי סימולציה (עקומת J / תזרים מצטבר) המדמה חברת גיוס תרומות המפעילה קמפיינים
דיגיטליים ממומנים עבור עמותה, ומשווה **4 מסלולים מסחריים** על פני **96 חודשים**.

קובץ יחיד (`index.html`) — vanilla JS, ללא build, ללא שרת. עובד מכל מקום:
GitHub Pages, Netlify, או הטמעה ב-`iframe` / HTML widget בכל CMS.

## הרצה מקומית
פותחים את `index.html` בדפדפן. זהו.

## המסלולים
1. **תקורות + עמלת גיוס** — עמלה על גיוס/המרות/חידושים, פחות קיזוז ביטולים.
2. **עמלה בניכוי ביטול** — עמלה על גיוס בניכוי % מוסכם; ביטולים לא משפיעים על העמלה.
3. **סליקה בלבד** — עמלה כ-% מהגבייה החודשית בפועל.
4. **אחוז כולל (הו"ק+ח"פ)** — עמלה כ-% מהגבייה בפועל, מחזור תרומה ארוך יותר.

## מה שופר מול הגרסה המקורית (v33 → v34)
- **שיתוף קישור** — הכפתור "🔗 שתף" קודד את כל הפרמטרים ל-`#s=...` בכתובת,
  מעתיק ללוח ומציג טוסט. במקור הפונקציה לא הייתה קיימת כלל.
- **שמירה אוטומטית** — המצב נשמר ל-`localStorage` ונטען ברענון.
- **תיקון קריסה** — `baseCols` היה לא-מוגדר וגרם לקריסת הטבלה בעת הפעלת מודל הפסקה.
- **ניקוי באג** — `var isStop=isStop;` (השמה עצמית → undefined) הוחלף בערך מפורש,
  בשמירה על התנהגות וזהות מספרית מלאה מול האתר המקורי.

## פרסום (GitHub Pages)
מופעל מענף `main`, נתיב `/`. הקובץ `index.html` מוגש כפי שהוא.
